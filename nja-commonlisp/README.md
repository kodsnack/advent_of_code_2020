# Advent of Code 2020

My solutions to [Advent of Code 2020](https://adventofcode.com/2020/)
using Common Lisp.

# Setup

Install [a Common Lisp implementation](https://common-lisp.net/implementations)
and [QuickLisp](https://www.quicklisp.org/).
